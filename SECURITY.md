# Security Policy

## Supported Versions

Please see the table below for versions that are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover any security-related issues, please report them by opening a private vulnerability report via GitHub Security Advisories for this repository. 

Alternatively, if private reporting is unavailable, you can open an issue detailing the context of the vulnerability, or contact the repository owner directly. We take security vulnerabilities very seriously and will work to resolve the issue promptly.
